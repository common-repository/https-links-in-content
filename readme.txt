== Description ==

When WP is opened through HTTPS, all local absolute links in content are replaced from HTTP to HTTPS.<br>
Additional hosts can be specified through admin interface (like graph.facebook.com which works on HTTP and on HTTPS).<br>
Options HOME and SITEURL shuld be changed to HTTPS when enabling this plugin.
